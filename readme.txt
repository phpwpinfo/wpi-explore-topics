Lay all your tags in alphabetical order on a page and search as you type (ajax) filter for a topic/tag
== Description ==

This plugin automatically lays all your tags in alphabetical order on a page and adds an ajax filter input box which allows to search as you type for a particular topic/tag
It is a great way to show all the topics your site covers on various areas(keywords)

**Features**
1. Search as you type
2. Supports Multisite
3. Simple and easy to customize

Here is a demo: http://phpwpinfo.com/explore-topics/




== Installation ==

1. Upload the plugin files in the same way you'd install any other plugin.
2. Activate the plugin
3. Create a page and add shortcode [WPI_EXPLORE_TOPICS]
